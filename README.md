# Week 2 Task 1 - Data Cleaning and Exploratory Data Analysis

## Dataset Used

Titanic Dataset

## About the Task

In this task, I worked with the Titanic dataset to understand the process of data cleaning and exploratory data analysis (EDA). The aim was to identify patterns in the data and gain insights using Python libraries.

## What I Did

* Loaded the dataset into Google Colab using Pandas.
* Checked the dataset structure and summary statistics.
* Identified missing values in different columns.
* Filled missing values in the Age column using the median.
* Filled missing values in the Embarked column using the mode.
* Removed the Cabin column because it contained too many missing values.
* Created visualizations to understand the data better.
* Generated a correlation heatmap to study relationships between numerical features.

## Observations

* Female passengers had a higher survival rate than male passengers.
* Passengers travelling in first class had a better chance of survival.
* Most passengers were between 20 and 40 years of age.
* Age showed only a weak relationship with survival.
* Passenger class and fare had a stronger influence on survival.
